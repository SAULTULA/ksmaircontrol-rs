# KSM AirControl 🎙️🚀

![KSM AirControl](fondo.png)

**KSM AirControl** es una avanzada e integral solución de automatización radial y centro de control diseñada para estudios de transmisión, broadcasters y creadores de contenido que buscan una herramienta potente, moderna y altamente confiable.

Esta aplicación marca el primer gran hito de **KSM Servicios (Creador de Aplicaciones y Soluciones Informáticas)**, representando meses de arduo trabajo, ingeniería de software detallada y un compromiso inquebrantable con la calidad.

---

## 🌟 Características Principales

- **Automatización Profesional:** Gestión fluida de listas de reproducción (Playlists) con capacidades de auto-dj, detección automática de puntos de mezcla (Cue/Mix) y transiciones perfectas sin silencios.
- **Cartwall Interactivo:** Matriz de efectos de sonido y jingles de acceso rápido (Hotkeys), ideal para intervenciones en vivo y programas dinámicos.
- **Codificador (Encoder) Integrado:** Transmisión directa a servidores Icecast/Shoutcast sin necesidad de software de terceros.
- **Cerebro Inteligente & Telemetría:** Monitoreo en tiempo real del estado de la estación, sincronización con la nube (Supabase) y capacidades de asistencia automatizada.
- **Diseño Adaptativo y Moderno:** Una interfaz de usuario oscura, cuidada hasta el más mínimo detalle (BEM, CSS Modules), diseñada para reducir la fatiga visual durante largas horas de transmisión.
- **Multimonitor y Paneles Flotantes:** Posibilidad de extraer componentes clave (como el Cartwall y la Playlist) a ventanas independientes para aprovechar configuraciones de múltiples pantallas.
- **Sistema de Licencias Seguro:** Validación por hardware (HWID) que garantiza un despliegue controlado y seguro para clientes comerciales.

## 🛠️ Tecnologías Utilizadas

KSM AirControl ha sido construido utilizando un stack tecnológico moderno y robusto:
- **Electron:** Para garantizar un rendimiento nativo de escritorio multiplataforma.
- **React.js & Vite:** Para una interfaz de usuario reactiva, modular y de altísimo rendimiento.
- **Audio Web API / Howler.js:** Para el procesamiento de audio de baja latencia.
- **Supabase:** Base de datos y telemetría en tiempo real.
- **FFmpeg & SQLite:** Análisis ultrarrápido de formas de onda y escaneo de metadatos de bibliotecas musicales masivas.

KSM AirControl cuenta con un sistema de **Actualización Automática**. Una vez instalado, el sistema revisará periódicamente este repositorio en busca de nuevas mejoras.
